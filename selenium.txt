1. Ставим Python на комп. Во время установки смотрим, чтобы была включена настройка Add Python 3.x to PATH.

> Проверить, правильно ли всё установилось, можно командой python --version в командной строке;

2. Создаем папку виртуального окружения, переходим и создаем само окружение:

> mkdir environments
> cd environments
> python -m venv selenium_env

3. Активируем окружение:

> selenium_env\Scripts\activate.bat

> Убедиться, что всё работает, можно по наличию (selenium_env) в начале строки;
> Команду нужно выполнять при каждом запуске командной строки;

4. Запускаем интерпретатор python командой python.

==========================================================================================================

В командной строке переходим в папку с окружением (cd c:\environments) и выполняем команду pip install selenium==3.14.0

1. Скачиваем chromedriver (или любой другой web driver для нужного браузера). Создаем папку, копируем туда exe-файл;

2. В свойствах Компьютера выбираем "Дополнительные параметры системы" > таб "Дополнительно" > кнопка "Переменные среды". В списке находим переменную PATH - открываем её. В правой части 
окошка жмем "Добавить", и добавляем путь к нашему вебдрайверу (например, "c:\chromedriver");

3. Чтобы проверить, что всё работает, в командной строке выполняем path и видим, что добавенный путь отображается в ответе. Также можно выполнить запрос chromedriver, чтобы убедиться, что 
вебдрайвер работает корректно;
